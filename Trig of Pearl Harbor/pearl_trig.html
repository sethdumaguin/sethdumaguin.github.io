Trig of Pearl Harbor


I made this game during my senior year in Cathedral High School. I programmed this game using HTML5, CSS, and JavaScript. I drew all the artwork using Adobe Illustrator.
The purpose of this game is to give students a way to practice their trigonemetric functions and unit circle values. I wanted to make the game more interesting so I decided to let the game teach something about the history. This is why the setting of the game is in Pearl Harbor. The player takes the role of a fighter jet pilot collecting care packages during the attack on Pearl Harbor, but there is a catch though. Each care package is labeled with an answer and the player must pick the correct answer.
This game was entered to compete in the Globaloria Globey Awards and made it to the national finals, finishing in the top 5.
